# 🏡 나중사 (나를 위한 중개사)

## 프로젝트 소개

부동산 매물 요청자와 중개사를 연결하는 **부동산 매칭 플랫폼**입니다.

사용자는 원하는 매물 조건을 등록하고  
중개사는 해당 요청에 맞는 매물을 제안할 수 있습니다.

---

## 성능 개선 경험

### QueryDSL 기반 쿼리 최적화

기존에는 복잡한 조건 검색에서 성능 저하가 발생했습니다.

이를 해결하기 위해 QueryDSL 기반 동적 쿼리 구조로 변경하여

- DB 성능 약 **30% 개선**
- API 응답 속도 **약 400ms 단축**

을 달성했습니다.

---

## 비용 절감 경험

### 안심번호 발급 구조 개선

기존 구조에서는 불필요한 안심번호 발급이 발생했습니다.

발급 조건을 개선하여

- 불필요한 발급을 줄이고
- **운영 비용 약 40% 절감**

효과를 얻었습니다.
